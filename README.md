\# Java Combo Verifier



Programa desenvolvido em Java para validar a contratação de combos de serviços de telecomunicações.



\## 🚀 Funcionalidade

O sistema recebe uma lista de serviços contratados e verifica se o cliente possui o \*\*Combo Completo\*\*, que consiste em:

\- Móvel

\- Banda Larga

\- TV
